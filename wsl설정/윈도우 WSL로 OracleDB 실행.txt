# 윈도우 WSL 설치
1. 시작 >> 검색 powershell 관리자 권한으로 실행
2. powershell에서 다음을 시행하여 wsl 설치 및 설정 
wsl --install
wsl --set-default-version 2

# Docker Desktop 설치 
1) https://www.docker.com/products/docker-desktop/로 접속 
2) Download Docker Desktop 누른 후 알맞은 제품 다운로드(Download for windows for AMD64) 
3) Docker Desktop 설치 
4) 윈도우 로그아웃 후 로그인 또는 재시작 

# 도커데스크탑 실행
1) 바탕화면의 도커데스크탑 실행
2) 탐색기에서 별도의 경로에 oracledb.yaml 파일 생성(예: c:\oracle_install)
------------------------------------
services:
  oracle21c:
    image: cielciel/myoracle21:1.0
    restart: always
    container_name: oracle21c
    shm_size: '2gb'
    ports:
      - "1521:1521"
      - "5500:5500"
    volumes:
      - ./db/oracle/data:/opt/oracle/oradata
    environment:
      ORACLE_PASSWORD: oracle
      TZ: Asia/Seoul
    privileged: true
----------------------------------------------

3) 현재 경로에서 우클릭 >> 터미널실행으로 Powershell 실행 한후 다음과 같이 docker 설치 확인
docker --version
4) 다음 명령으로 db 컨테이너 생성(시간 소요됨) 
docker-compose -p db -f oracledb.yaml up -d 
5) DB 컨테이너 실행 확인 및 접속
docker ps 
docker exec -it oracle21c bash
6) 이후 과정은 오라클 설치 완료 모니터링 및 SQLDeveloper 실행 후 예제 데이터 생성하는 과정이므로 교안의 설치과정 참조
